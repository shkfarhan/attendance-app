# Employee Attendance System

A production-ready, secure employee attendance application built with Next.js and Firebase.

## Features
- **Strict Location Fencing**: Employees must be within 100m of the office.
- **Server-Side Validation**: All time and distance calculations happen on the server.
- **Rules Enforcement**: Late-in equals Late-out policy auto-calculated.
- **Role-Based Access**: Separate Employee and Admin dashboards.
- **Secure**: Firestore Security Rules and Server Actions prevent tampering.
